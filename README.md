# LocalBiz
`LocalBiz` 的是 Local Business 的缩写, 含义为着附近的商业.  
此项目的目标为让用户可以足不出户的体检附件的商业服务

## 第三方库
1. provider: 状态管理
2. json_annotation: JSON序列化与反序列化
3. dio: 发送HTTP请求
4. shared_preferences: 数据持久化
5. go_router: 路由
6. animations: 动画
7. logger: 日志

## 运行项目
1. 安装 **Flutter** 运行环境 参考[**官网**](https://docs.flutter.dev/get-started/install)
2. 运行项目 `flutter run`
